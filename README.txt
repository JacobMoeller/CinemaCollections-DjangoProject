This Django Project allows the user to create an account and view a list of movies and blog posts.

Movies and blog posts can only be viewed if the user has created an account.

Admins can add movies and blog posts via the admin page.

Admin Login:

	Username: admin

How to run:

	1. cd into djangoProject/venv/django-SE-final/mysite

	2. Run command: python manage.py runserver

	3. Open up a browser and go to this link: http://127.0.0.1:8000/
